# 智能协同云图库后端开发日志


## 技术栈
- **后端框架**: Spring Boot, MyBatis Plus
- **数据库**: MySQL, Redis
- **对象存储**: 腾讯云 COS
- **安全**: JWT 认证, RBAC 权限控制
- **API 文档**: Swagger
